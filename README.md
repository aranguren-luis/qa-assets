# qa-assets
Imágenes y capturas utilizadas en los proyectos de control de calidad, organizadas por carpetas para facilitar el seguimiento de incidencias.
---


1. **PicGo** está configurado para subir imágenes a este repositorio.
2. Al tomar una captura o copiar una imagen al portapapeles, se sube automáticamente.
3. La URL de la imagen se genera en formato Markdown y se copia al portapapeles.
